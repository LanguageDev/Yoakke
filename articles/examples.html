Examples Overview

This directory contains comprehensive examples showing how to use SynKit components.

Available Examples

Basic Examples

  • Simple Lexer - Basic token recognition
  • Expression Parser - Mathematical expressions with precedence
  • Calculator - Complete calculator with error handling

Advanced Examples

  • JSON Parser - Full JSON parsing with proper error reporting
  • C Preprocessor - Using the built-in C syntax components
  • Custom Language - Building a small domain-specific language

Integration Examples

  • Error Reporting - Rich diagnostic output with syntax highlighting
  • AST Generation - Building abstract syntax trees
  • Code Generation - Generating code from parsed input

Running Examples

All examples are located in the SynKit/Examples/ directory and can be run with:

cd SynKit/Examples/Lexer.Sample
dotnet run

Example Structure

Each example follows this structure:

  • README.md - Overview and usage instructions
  • Program.cs - Main entry point
  • TokenType.cs - Token definitions (if applicable)
  • Parser.cs - Parser implementation (if applicable)
  • sample-input.txt - Example input files

Learning Path

  1. Start with Simple Lexer - Learn basic tokenization
  2. Move to Expression Parser - Understand parsing concepts
  3. Try Calculator - See complete error handling
  4. Explore JSON Parser - Study a real-world grammar
  5. Build Custom Language - Apply concepts to your domain
